DUMMIES WEB APPLICATION

This project is step by step Web Application creation tutorial for beginners. In this tutorial you will learn the basic of HTML5, CSS3, Vanilla JavaScript.

OUR TOOLS

For web development, you need editor to create the file for the website and a browser to view your website.

  • Install Visual Studio Code- IDE.
  • Install Chrome

Creating an HTML Document

Let’s create the first HTML document for our website.

  1. Create folder on your computer for your project. Name the folder dummies (or anything want)
  2. Open VS Code
  3. Open the File menu select Open Folder and select the folder you created and open it.
  4. Right click below the folder and select New File. Name the file index.html.
  5. Now you have blank text file named index.html

Right click on your index.html (the file below the dummies folder). Then click open with Browser. A browser window should open up and show your first website.
